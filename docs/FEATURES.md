# Feature Set

A tool for a person who cares about the future of their money, family, and spouse — not an accountant's tool. Every feature earns its place by either (a) reducing the friction that stops 43% of people from ever starting, (b) covering a document the authoritative checklists say a complete plan needs, or (c) getting the user across the finish line to a *validly executed* plan.

## Product principles

1. **Momentum over completeness** — the user should feel progress in the first five minutes. Guided steps, a visible readiness score, never a wall of forms.
2. **Plain language** — every legal term gets a one-line explanation. No jargon without a tooltip.
3. **The plan is bigger than the will** — beneficiary designations, POAs, directives, and the asset inventory are peers of the will, not add-ons.
4. **Done means signed** — a drafted document that was never witnessed is a failed outcome. Execution guidance is a first-class feature.
5. **UPL-safe by design** — forms and information, never individualized advice. Attorney-reviewed templates, prominent disclosures, referral (not advice) pathways.

## P0 — Core (the reason the product exists)

| Feature | Description |
|---|---|
| **Household & people setup** | You, spouse/partner, children, other loved ones. Drives everything else (guardianship needs, joint vs individual planning, beneficiary candidates). |
| **Asset & liability inventory** | Accounts, real estate, insurance, retirement, digital assets, debts. Estimated values → net-estate view. The foundation for beneficiary tracking and probate-avoidance insight. |
| **Estate readiness dashboard** | A single score + checklist showing what's done, what's missing, and the next best action. The anti-procrastination engine. |
| **Guided will interview** | Plain-language, state-aware interview → last will & testament: executor (with bond-waiver and powers boilerplate), guardianship for minor children, specific gifts, residuary estate. Attorney-reviewed templates per state. |
| **Beneficiary designation tracker** | Track designations on life insurance/401(k)/IRA/TOD accounts and flag conflicts with the will (designations override wills — the classic layman trap). |
| **Financial power of attorney + advance healthcare directive** | The two documents that matter while you're alive. State-specific forms, agent selection with plain-language explanation of powers. |
| **Execution walkthrough** | Per-state signing requirements (witness count, disinterested-witness rules, notarization), printable signing-day checklist, and a "mark as executed" confirmation step. Differentiator: no incumbent solves this. |
| **UPL compliance layer** | "Not a law firm / not a substitute for an attorney" disclosures at generation and download points; NC-consent-judgment patterns throughout. |

## P1 — The complete plan

| Feature | Description |
|---|---|
| **Revocable living trust** | Guided creation + **funding guidance** (retitling assets into the trust — the step everyone skips). Probate-avoidance explainer tied to the user's actual asset list. |
| **HIPAA authorization** | Companion to the healthcare directive. |
| **Guardianship deep support** | Backup guardians, letters of intent for care preferences. |
| **Document vault** | Encrypted storage of executed documents + pointers to originals ("safe deposit box at ..."). |
| **Executor / family sharing** | Invite executor, spouse, or trusted people with scoped access (e.g., executor sees documents + asset list, not values until death). |
| **Digital assets & final wishes** | Account credentials inventory (or pointer to password manager), funeral/memorial preferences, letters to loved ones. |

## P2 — The living plan

| Feature | Description |
|---|---|
| **Life-event review triggers** | Marriage, birth, divorce, death, move (state change re-validates documents!), retirement, job change — prompt targeted plan updates; periodic 3-year review nudge. |
| **Probate-avoidance insights** | Per-asset "does this pass through probate?" flags based on titling/designations. Information, not advice. |
| **Estate tax awareness** | Federal exemption + state estate/inheritance tax flags based on state + net estate. (Blocked on verifying current figures.) |
| **Attorney referral pathway** | For complex situations (blended families, special-needs dependents, business owners, Louisiana), hand off gracefully. |
| **Remote execution (where legal)** | E-will / remote-online-notarization support in states that allow it. (Blocked on fresh state-by-state research.) |

## Explicit non-goals (for now)

- Louisiana documents (civil-law regime; every major competitor also excludes it — revisit as a stretch differentiator)
- Individualized legal or tax advice of any kind
- Advisor-facing / B2B features (wealth.com territory)
- Irrevocable trusts, complex tax-minimization vehicles (attorney referral instead)
